# Пример выполнения РГР по ЧМ


Содержит расчетную часть для формирования отчета по задачам в pdf на markdown и tex.



## Вам понадобится


 - NodeJS(v0.10.35)

 - [Pandoc](https://pandoc.org/)(1.13.2)
 
- [MiKTeX](https://miktex.org/)(XeTeX, Version 3.1415926-2.5 (MiKTeX 2.9 64-bit))

 - [JavaScript Lint](http://www.JavaScriptLint.com)(0.3.0)
 - В задаче 3.2 Кубический сплайн, для упрощения выражений вызывается ядро mathematica
 - Для проверки текста на ошибки используется сервис и соответствующий скрипт на perl от [yandex](https://tech.yandex.ru/speller/)
 - ну и методическое пособие...
